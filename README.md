# 🏛️ Malaysia History · Escape Room

An interactive educational browser game built with **HTML, CSS, and JavaScript**, where players explore Malaysian history while escaping through 10 challenge rooms.

---

## 🇲🇾 About the Game

**Malaysia History · Escape Room** is a survival-style learning game set in a fictional escape-room environment. Players collect historical knowledge orbs while avoiding enemies across multiple rooms. Each orb reveals a real fact about the history, culture, and development of 🇲🇾 Malaysia.

The goal is simple:  
👉 Collect all orbs in each room  
👉 Survive enemies  
👉 Escape all 10 rooms

---

## 🎮 Features

- 🧠 **100+ Malaysian history facts** embedded into gameplay  
- 🏃‍♂️ 10 progressively structured rooms  
- 👾 Enemy AI that actively chases the player  
- ❤️ Lives system (5 hearts per run)  
- ⭐ Star rating based on performance  
- 🔊 Dynamic sound effects + background music  
- 📜 Educational learning integrated into gameplay  
- 🎨 Canvas-based smooth 2D rendering  
- 📱 Keyboard controls (WASD / Arrow Keys)

---

## 🎯 How to Play

### Objective
Collect all **10 orbs** in each room to unlock the next stage while avoiding enemies.

### Controls
| Action | Key |
|--------|-----|
| Move Up | W / ↑ |
| Move Down | S / ↓ |
| Move Left | A / ← |
| Move Right | D / → |

### Rules
- Each orb gives a **real Malaysian history fact**
- Touching an enemy reduces **1 life**
- Lose all lives = Game Over
- Complete all 10 rooms to win
